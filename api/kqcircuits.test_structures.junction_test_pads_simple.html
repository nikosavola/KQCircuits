kqcircuits.test_structures.junction_test_pads_simple¶

class kqcircuits.test_structures.junction_test_pads_simple.JunctionTestPadsSimple[source]¶

Bases: kqcircuits.test_structures.junction_test_pads.JunctionTestPads

Junction test structures.

Produces an array of junction test structures within the given area. Each structure consists of a SQUID, which is connected to pads. There can be either 2 or 4 pads per SQUID, depending on the configuration. Optionally, it is possible to produce only pads without any SQUIDs.
