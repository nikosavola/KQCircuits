kqcircuits.chips.junction_test2¶

class kqcircuits.chips.junction_test2.JunctionTest2[source]¶

Bases: kqcircuits.chips.chip.Chip

The PCell declaration for a JunctionTest2 chip.

produce_impl()[source]¶

This method builds the PCell.

Adds all refpoints to user properties and draws their names to the annotation layer.

PCell parameters:

  • pad_width (Double) - Pad Width, default=500, unit=[μm]

  • junctions_horizontal (Boolean) - Horizontal (True) or vertical (False) junctions, default=True

  • squid_type (String) - SQUID Type, default=QCD1, choices=['NoSquid', 'QCD1', 'QCD2', 'QCD3', 'Sim', 'SIM1']

  • pad_spacing (Double) - Spacing between different pad pairs, default=100, unit=[μm]
