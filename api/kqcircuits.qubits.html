kqcircuits.qubits¶

  • kqcircuits.qubits.qubit
  • kqcircuits.qubits.swissmon

Module contents¶

PCell declaration classes for qubits.

The qubits contain a SQUID, which can either be code-generated or loaded from a manual design. The qubits can also have a fluxline.
