kqcircuits.util.parameters¶

kqcircuits.util.parameters.add_parameters_from(cls, *args)[source]¶

Decorator function to add parameters to the decorated class.

If args is empty it takes all parameters of cls, otherwise only takes parameters mentioned in args. Only add parameters that are not already class attributes.

Parameters

  • cls – the class to take parameters from

  • args – is an optional list of parameter names to take
